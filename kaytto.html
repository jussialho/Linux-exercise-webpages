NetBeansin peruskäyttö

NetBeans IDE on ohjelmistojen kehitysympäristö, eli ohjelmointityökalu, jolla voidaan koodata, kääntää, ajaa ja debugata sovelluksia. Se tukee suurinta osaa ohjelmointikielistä ja tässä ohjeessa esitellään yksinkertaisen Java-sovelluksen tekeminen ja kääntäminen.

NetBeansin käynnistys

Ohjelma voidaan käynnistää sekä komentoriviltä (Terminal) että sovellusvalikosta (Show Applications). Kannattaa huomata, että se on nimellä Apache NetBeans. Terminalista NetBeans IDE käynnistetään komennolla

netbeans
.

Ohjelman käynnistyttyä aukeaa aloitussivu (Start Page), mistä löytyy esim. viimeisimmät projektit ym. Tärkeä painike on Install Plugins, josta pääsee lataamaan ja päivittämään ohjelman lukuisia lisäpalikoita.

Ensimmäinen Java-sovellus

NetBeans IDE:llä voidaan tehdä käytännössä mitä ohjelmointiprojekteja tahansa, mutta sitä käytetään paljon varsinkin Java-sovelluksien tekemisessä. Katsotaan seuraavaksi miten tehdään ja käännetään yksinkertainen Java-sovellus.

  1. Valitaan File-valikosta New Project....

  1. New Project -ikkunassa valitaan aluksi projektin kategoria. Koska teemme nyt yksinkertaisen Java-ohjelman, valitsemme Java with Ant.

    Lisäksi pitää valita projektin tyyppi. Valitsemme nyt Java Application ja siirrymme eteenpäin klikkaamalla Next.

  1. New Java Application -ikkunassa annetaan projektille nimi ja tallennushakemisto. Nimi kannattaa valita mahdollisimman kuvaavaksi, mutta projektin tallennuspaikkaa ei välttämättä tarvitse muuttaa. Kannattaa rastittaa valinta Create Main Class, jotta Java-ohjelman main-luokka tehdään valmiiksi.

    Kun kaikki on valmista, klikataan Finish

  1. Hetken päästä pitäisi avautua itse ohjelmointi-ikkuna. Se on jaettu valikoiden ja työkalurivin lisäksi kolmeen osaan. Vasemmalla on avoinna olevan projektin hallintaikkuna, josta löytyy mm. kaikki projektin kirjastot, tiedostot, luokat ja metodit. Oikealla on suurempi editointi-ikkuna koodin kirjoittamista varten omalla välilehdellään. Välilehdiltä löytyy myös Start Page ja siellä voi luonnollisesti olla auki useampiakin editointi-ikkunoita.

  1. Testataan vielä Java-tulkin ja -kääntäjän toiminta kirjoittamalla main -funktion sisään koodi, jolla saadaan ohjelma tulostamaan yhden rivin tervehdys näytölle :).

    Kirjoiteaan riville 19 koodi System.out.println("Moikka maailma!");.

    Ajetaan ohjelmalla valitsemalla valikosta Run | Run Project, painamalla näppäimistöltä F6 tai valitsemalla työkaluriviltä vihreä Run Project -painike .

    Ohjelma voidaan myös debugata valitsemalla Debug Project.

    Jos koodi on syntaksiltaan oikein ja NetBeans IDE toimii kuten pitäisi, tulee ohjelmaikkunan alareunaan vihreä teksti onnistuneen ajon merkiksi ja ohjelma tulostaa yhden rivin tekstiä, mikä oli tarkoituskin.

Kannattaa huomata, että NetBeans IDE on paljon muutakin kuin pelkkä koodieditori. Se sisentää koodirivejä automaattisesti, ehdottaa funktioita syötteen perusteella, lisää automaattisesti sulkeita, korostaa syntaksivirheitä ja tekee paljon muutakin hyödyllistä. Lisäksi siinä on paljon hyödyllisiä työkaluja suuremmankin projektin hallintaan ja se tukee kaikkia yleisimpiä ohjelmointikieliä.
